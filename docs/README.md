# iDryer Link — краткое руководство

Link — это модуль связи для iDryer. Он подключается в Ethernet‑порт контроллера и через него общается с платой управления (порт используется как разъем с питанием и UART, не как сетевой). Link выводит сушилку в интернет и связывает ее с portal.idryer.org.

## Как подключить к контроллеру 
1) Отключите питание контроллера.
2) Соберите кабель RJ45, сверяйтесь с ![RJ45](./img/RJ45.png), чтобы пары не перепутать. Важное: RJ45 здесь — это просто разъем питания/UART, не включайте его в сетевой коммутатор.
3) Соедините провода с esp32-C3 Super mini по схеме ![esp32superMini](./img/esp32superMini.png)

## Как прошить через веб‑флешер
Веб‑флешер живет на https://install.idryer.org/ 

- Подключите Link к USB порту компьютера.
- Откройте [страницу](https://install.idryer.org/ ) и выберите устройство **iDryer Link**.
- В выпадающем списке выберите плату:
   - `ESP32-C3 super-mini` — основной вариант для серийных модулей.
   - `ESP32-C3 DevKit` — если у вас отладочная плата.
- Нажмите **Connect**, выберите серийный порт (обычно `USB JTAG/serial` или `CH340`). Если прошивка не стартует, зажмите `BOOT` на плате и кратко нажмите `RST`.
- Нажмите **Install**. Флешер сам прошьет три файла (bootloader.bin, partitions.bin, firmware.bin) по смещениям из манифеста.
- После 100% откроется Improv‑мастер: введите SSID и пароль Wi‑Fi, дождитесь статуса “Connected”.
- Если Improv‑мастер не открылся, отключите USB и повторите подключение, выбрав **Connect** без повторной прошивки.
- Нажмите кнопку **Start Claim**. Появится строка `PIN:123456` — это PIN, действует ~5 минут.
- Перейдите на https://portal.idryer.org → «Добавить устройство» → введите PIN. После успешной привязки устройство появится в списке.
- Отключите USB и подключите Link к контроллеру по RJ45.
- Включите питание iDryer
- об успешном соединении светодиодная индикация сообщит голубым "дыханием"

## Что должно получиться
- Контроллер видит Link сразу после подачи питания.
- В веб‑портале устройство появляется онлайн в течение 1–2 минут после подключения к Wi‑Fi.
- Если статус не появляется: перепроверьте распиновку по `img/RJ45.png`, качество обжима и правильность выбора платы в шаге прошивки и конфигурацию портов с меню iDryer.
