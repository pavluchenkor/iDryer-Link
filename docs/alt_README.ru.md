# iDryer Link

Link — модуль связи для iDryer. Подключается в RJ45-порт контроллера и выводит сушилку в интернет через портал [portal.idryer.org](https://portal.idryer.org/).

!!! note ""
    RJ45 здесь используется как разъём питания и UART — не как сетевой интерфейс. Не подключайте Link в сетевой коммутатор.

## Подготовка к подключению

!!! warning ""
    Перед подключением Link к контроллеру **отключите питание контроллера**.

---

## Сборка кабеля RJ45

1. Соберите кабель RJ45, сверяясь со схемой:

    ![RJ45](./img/RJ45.png)

2. Подключите провода к ESP32-C3 по схеме:

    ![ESP32-C3 Super Mini](./img/esp32superMini.png)

    ```
    UART_RX_PIN 6 (бело-синий)
    UART_TX_PIN 7 (бело-зелёный)
    ```

    !!! note ""
        Расположение контактов 6 и 7 может отличаться в зависимости от производителя платы. Сверьтесь с пинаутом вашего модуля.

## Прошивка через веб-флешер

1. Подключите Link к USB-порту компьютера.
2. Откройте [install.idryer.org](https://install.idryer.org/) и выберите устройство **iDryer Link**.
3. Выберите плату:
   - `ESP32-C3 super-mini` — для серийных модулей.
   - `ESP32-C3 DevKit` — для отладочных плат.
4. Нажмите **Connect**, выберите серийный порт (`USB JTAG/serial` или `CH340`).
5. Если прошивка не стартует — удерживайте `BOOT` и кратко нажмите `RST`.
6. Нажмите **Install**. Флешер прошьёт всё автоматически.
7. После 100% откроется Improv-мастер: введите SSID и пароль Wi-Fi, дождитесь статуса «Connected».

---

## Привязка к порталу

1. Нажмите **Start Claim**. На экране появится `PIN:123456` — действует ~5 минут.
2. Перейдите на [portal.idryer.org](https://portal.idryer.org/) → «Добавить устройство» → введите PIN.
3. После успешной привязки:
   - Отключите USB.
   - Подключите Link к контроллеру через RJ45.
   - Включите питание iDryer.
4. Успешное соединение — голубое «дыхание» светодиода Link.

---

## Ожидаемый результат

- Контроллер видит Link сразу после подачи питания.
- Устройство появляется онлайн на портале в течение 1–2 минут.

Если устройство не появляется — проверьте:
- Распиновку кабеля по `img/RJ45.png`.
- Качество обжима разъёма.
- Конфигурацию порта в меню iDryer (`GLOBAL → PORT CONFIG` → нужный порт должен быть `LNK`).
